<Steps>
  <Step title="Nhập email tài khoản">
    Nhập email liên kết với tài khoản Vero OMS rồi chọn **Send Code**.
  </Step>
  <Step title="Kiểm tra inbox">
    Lấy one-time recovery code được gửi tới email tài khoản.
  </Step>
  <Step title="Nhập recovery code">
    Nhập code rồi chọn **Verify & Continue**.
  </Step>
  <Step title="Tiếp tục vào Vero OMS">
    Code hợp lệ sẽ khôi phục truy cập và mở application workspace.
  </Step>
</Steps>
