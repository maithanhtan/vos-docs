## Order Blotter

![Orders blotter](/images/vero-oms-ui/orders-blotter.png)

Bảng Orders liệt kê account, symbol, side, quantity, price, status và thông tin thời gian cho từng lệnh.

| Control | Hành vi |
|---|---|
| Status / side filters | Lọc blotter theo trạng thái lệnh và side. |
| Account và symbol filters | Lọc theo một account hoặc một symbol; `ALL` là trạng thái reset. |
| From / To date-time | Cập nhật range hiển thị và shared order time range. |
| Export | Tải các dòng đang hiển thị/sort ra file XLSX. |
| Header sort | Sort theo id, date/time, account, symbol, side, quantity, fill %, price, market hoặc status. |
| Row click | Chọn lệnh và cập nhật panel chi tiết bên phải. |
| Checkbox / Select all | Chọn một hoặc nhiều dòng cho bulk action. |

Khi có dòng được chọn, bulk action bar hiển thị số dòng chọn, số lệnh có thể cancel và estimated notional. `MODIFY ORDER` chỉ bật khi có đúng một lệnh cancellable. `CANCEL ALL` mở confirmation để cancel remaining working quantity.

Panel chi tiết bên phải hiển thị order summary, latency breakdown, child fills và action liên quan tới lệnh đang chọn.

| Detail action | Hành vi |
|---|---|
| Parent link | Mở order-chain modal khi lệnh có quan hệ parent hoặc child. |
| Modify | Mở dialog cho phép sửa price hoặc quantity, không sửa cả hai trong cùng request. |
| Cancel Order | Mở confirmation cho remaining working quantity. |
| Latency breakdown | Load timing details của lệnh đang chọn và hiển thị latency breakdown. |

Modify và cancel bị disable khi lệnh đang chọn không ở trạng thái working, pending hoặc armed có thể thao tác.
