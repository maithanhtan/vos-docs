## Latency Overview

Latency overview chứa bộ lọc, KPI latency, chart latency vs throughput và system load.

![System latency overview](/images/vero-oms-ui/system-latency-overview.png)

| Latency control | Hành vi |
|---|---|
| Range selector | Chuyển preset window và reset manual range state. |
| From / To date-time | Set manual latency range. |
| Type / account / status | Lọc latency rows theo message type, account và status. |
| Symbol / order search | Lọc rows theo symbol hoặc order id. |
| Auto refresh 3s | Bật/tắt periodic reload của latency report. |
| Chart zoom | Set manual range từ chart window được chọn. |

## Users

Users hiển thị bộ lọc user, bảng user, access overview, login history và selected user panel. Thông báo data unavailable xuất hiện khi không load được user information.

![System users](/images/vero-oms-ui/system-users.png)

Users workspace gồm text filter, bộ lọc trạng thái user, thao tác refresh và **NEW USER**. Khi chọn một dòng trong bảng, panel selected-user cập nhật identity, state, roles, allowed features, thời điểm tạo, last login, login IP và số active session.

![System users access overview](/images/vero-oms-ui/system-users-access-overview.png)

Mỗi dòng user có các action button để đổi state, reset password, reset 2FA và chỉnh access. Nút đổi state mở confirmation dialog activate/deactivate. Nút password mở dialog reset password và yêu cầu mật khẩu tối thiểu 8 ký tự. Nút 2FA mở confirmation trước khi xoá second-factor setup hiện tại.

| User action | Hành vi |
|---|---|
| Select user | Cập nhật identity, role, access, login và session detail. |
| Activate / Deactivate | Mở confirmation trước khi đổi account state. |
| Reset password | Yêu cầu password mới tối thiểu 8 ký tự. |
| Reset 2FA | Mở confirmation trước khi xoá second-factor setup hiện tại. |
| Edit access | Mở role và feature controls của user đang chọn. |
| New User | Mở form provision user. |

### Edit Access Dialog

**Edit access** có thể mở từ action button trong bảng hoặc từ selected-user panel. Các switch role và feature chỉ đổi local dialog state trước; **SAVE ACCESS** submit danh sách roles và allowed features đang chọn.

![System users edit access dialog](/images/vero-oms-ui/system-users-edit-access.png)

### Create User Dialog

**NEW USER** mở create-user dialog. Form gồm email, username tuỳ chọn, first name, last name, temporary password, role switches và feature switches. Nút hiện/ẩn password chỉ đổi trạng thái hiển thị của field. **CREATE USER** yêu cầu email và temporary password tối thiểu 8 ký tự.

![System users create user dialog](/images/vero-oms-ui/system-users-create-user.png)

## Audit

Audit hiển thị activity history, filter/search, refresh và panel chi tiết record đang chọn.

![System audit](/images/vero-oms-ui/system-audit.png)

| Audit control | Hành vi |
|---|---|
| Search all columns | Lọc audit entries đã load theo step, request, actor, detail và request id đang render. |
| Actor filter | Lọc activity theo actor khi refresh hoặc nhấn Enter. |
| Refresh | Reload audit entries. |
| Row click | Mở audit detail panel. |
| Related activity | Hiển thị timeline cho các entry cùng request id. |

Detail panel hiển thị step, request title, actor, role, action, affected object, request id, changed values và related activity khi có dữ liệu.
