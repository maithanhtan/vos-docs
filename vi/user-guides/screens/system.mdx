System cung cấp các view về latency, trạng thái service, user access và audit activity.

## Toàn màn hình

![System full page](/images/vero-oms-ui/system-full.png)

## Latency Overview

Latency overview chứa bộ lọc, KPI latency, chart latency vs throughput và system load.

![System latency overview](/images/vero-oms-ui/system-latency-overview.png)

## Latency Table

Bảng latency liệt kê từng order trace với gateway, risk, router, venue RTT, total latency và status.

![System latency table](/images/vero-oms-ui/system-latency-table.png)

## Order Latency Detail

Panel chi tiết bên phải hiển thị total latency, bottleneck, stage duration, trace metadata và order pipeline.

![System order latency detail](/images/vero-oms-ui/system-order-detail.png)

## Users

Users hiển thị bộ lọc user, bảng user, access overview, login history và selected user panel. Trạng thái service unavailable được ghi nhận như một error state.

![System users](/images/vero-oms-ui/system-users.png)

## Audit

Audit hiển thị audit trail, filter/search, refresh và panel chi tiết audit đang chọn.

![System audit](/images/vero-oms-ui/system-audit.png)
