## Command search

Ô `CMD>` dùng để tìm account hoặc group context. Khi chọn account hoặc group, workspace scope ở cạnh `NEW ORDER` được cập nhật; drawer, blotter, portfolio và risk view sẽ đọc theo context đó.

| Action | Kết quả |
|---|---|
| Nhập account, tên hoặc group | Mở suggestion theo account/group. |
| Chọn account | Set active account context cho trading và review page. |
| Chọn group | Set active group context và cập nhật chỉ báo `SHOWING GRP`. |
| Bấm New Order | Mở order drawer từ page hiện tại mà không đổi workspace. |

## Tìm module và action

Shortcut command có thể cấu hình sẽ mở dialog tìm các module và user action khả dụng.

| Action | Kết quả |
|---|---|
| Nhập tên module hoặc action | Lọc danh sách kết quả khả dụng. |
| Up / Down | Di chuyển active result. |
| Enter | Mở module hoặc chạy action đang chọn. |
| Escape | Đóng dialog mà không đổi workspace hiện tại. |
| Sign out | Kết thúc session hiện tại sau khi chọn. |

## Settings

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/settings-system-panel.png" alt="Settings system panel" />
  </div>
  <div className="guide-split-copy">

Settings có các tab `Profile`, `System`, `Hotkeys`, `Notifications`. Tab System chứa theme, accent color, UI scale, density, broker ticker source và clear cache.

| Khu vực setting | Hành vi |
|---|---|
| Profile | Hiển thị identity, current session, active sessions, account context và sign-out controls. |
| System | Cập nhật theme, accent color, UI scale, density, ticker source và local display preferences. Tab này cũng có action reset local cache. |
| Hotkeys | Liệt kê shortcut có thể cấu hình, nhận diện conflict và hỗ trợ reset defaults. |
| Notifications | Điều khiển notification channels, event types, presentation và sound preferences. |
| Close / backdrop / Escape | Đóng drawer mà không chuyển page. |

  </div>
</div>
