## PT Workspace Filters

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/put-through-filters.png" alt="Put Through filters" />
  </div>
  <div className="guide-split-copy">

Sidebar filter chứa symbol, side, status, account và IOI ID.

| Filter action | Hành vi |
|---|---|
| Symbol / side / status / account / IOI ID | Cập nhật draft filter set trong sidebar. |
| Matching IOIs | Hiển thị số advertisement match draft filter trước khi apply. |
| Apply Changes | Apply draft filters cho advertisements và PT orders. |
| Clear All | Reset workspace filters về trạng thái mặc định. |

  </div>
</div>

## Advertisements

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/put-through-advertisements.png" alt="Put Through advertisements" />
  </div>
  <div className="guide-split-copy">

Khu vực advertisements hiển thị tabs trạng thái, số IOI mở, matched và notional.

| Advertisement action | Hành vi |
|---|---|
| Status segmented control | Lọc advertisement board theo all, active, matched hoặc rejected. |
| Header sort | Sort visible IOIs theo id, time, symbol, side, quantity, price, transaction type, status hoặc party. |
| Row click | Chọn advertisement và pre-fill PT order form. |
| Pagination | Chuyển trang trong filtered advertisement list. |
| New Advertise | Mở dialog IOI advertisement với symbol, side, quantity, price, contact, market snapshot, notional và validation state. |

Dialog IOI advertisement chỉ gửi khi symbol, quantity và price hợp lệ. Chọn giá trong market snapshot sẽ fill vào price field.

  </div>
</div>

## PT order form

Form bên phải có thể nhập thủ công hoặc seed từ advertisement đang chọn.

| Field / action | Hành vi |
|---|---|
| Clear | Bỏ advertisement đang link và đưa form về fresh state. |
| Account / symbol | Set route và instrument. Chọn symbol cập nhật market snapshot và account summary. |
| Buy / Sell | Set quote side và tính lại notional. |
| Quantity / price | Nhận numeric input đã format và cập nhật validation. |
| Firm ID / Trader ID | Yêu cầu firm và trader pair hợp lệ từ reference data. |
| Place PT Quote | Gửi quote request khi account, symbol, quantity, price và counterparty ids hợp lệ. |
