Dashboard là workspace giao dịch chính. Màn hình này hợp nhất order entry, market depth, chart và order blotter.

## Tổng quan workspace

![Tổng quan Dashboard workspace](/images/vero-oms-ui/dashboard-full.png)

## Dashboard layout actions

| Action | Kết quả |
|---|---|
| Edit layout | Cho phép drag, resize và remove widget. |
| Add Widget | Mở widget picker cho chart, depth, ticket, blotter và các panel hỗ trợ. |
| Add All / Remove All | Thêm hoặc xoá toàn bộ widget khả dụng. |
| Share layout | Copy layout definition hiện tại. |
| Import layout | Mở dialog import JSON và chỉ apply sau khi validate. |
| Arrange / Reset | Sắp xếp lại hoặc khôi phục layout mặc định. |

## Widgets và context dùng chung

Widget picker chứa các panel giao dịch và giám sát khả dụng cho user hiện tại.

| Widget | Context vận hành |
|---|---|
| P&L Portfolio | Xem hiệu suất danh mục theo context tài khoản đang chọn. |
| Orders Workspace | Theo dõi lệnh active và historical orders. |
| Positions | Xem holdings và live mark-to-market values. |
| Risk Monitor | Xem cash, holdings, exposure và risk-check status. |
| Order Ticket | Chuẩn bị lệnh mà không rời Dashboard. |
| Reports | Xem reporting và audit summary khả dụng cho user. |
| Price Depth | Xem order book và recent trades của symbol. |
| Trading Chart | Xem candle và volume history của symbol. |
| Market Indices | Xem market index values khả dụng. |
| Price Board | Hiển thị market price table trong Dashboard layout. |
| Latency Analytics | Xem order timing và throughput theo khoảng thời gian. |
| Operational Workspace | Hiển thị account và operational view khi user có quyền. |

Các widget được link có thể dùng chung symbol hoặc account context. Unlink widget trước khi đổi context độc lập.

## Order Ticket

<div className="guide-split">
  <div className="guide-split-media">

  </div>
  <div className="guide-split-copy">

Ticket nhập lệnh chứa các field account, symbol, order type, side, price, quantity, order summary và buying power preview.

| Ticket action | Kết quả |
|---|---|
| Account / symbol search | Cập nhật ticket context và market preview. |
| Buy / Sell | Đổi side và tính lại order summary. |
| Order type | Đổi nhóm price field cần nhập. |
| Quantity stepper | Tăng hoặc giảm staged quantity. |
| Submit | Gửi ticket khi các field bắt buộc hợp lệ. |
| Cancel / reset | Xoá staged ticket state mà không submit. |

## Trading Chart

Chart hiển thị nến, timeframe, vùng giá và volume của symbol đang chọn.

![Dashboard trading chart](/images/vero-oms-ui/dashboard-trading-chart.png)

Đổi symbol sẽ reset chart context. Timeframe control đổi candle interval, còn load-more mở rộng lịch sử hiển thị khi có dữ liệu.

## TradingView mode

Chart hỗ trợ hai chế độ `CANDLE` và `TRADINGVIEW`. TradingView mode dành vùng chart cho embedded widget.

![Dashboard TradingView mode](/images/vero-oms-ui/dashboard-tradingview.png)

## Order Blotter

Order blotter dưới Dashboard cho phép lọc theo status, side, account, ngày giờ và hiển thị trạng thái lệnh live.

![Dashboard order blotter](/images/vero-oms-ui/dashboard-order-blotter.png)

User có thể chọn row để xem chi tiết. Header sortable dùng để sắp xếp visible set; action model đầy đủ của blotter được mô tả trong [Orders](/vi/user-guides/screens/orders).
