## Layout controls

![Layout toolbar](/images/vero-oms-ui/layout-toolbar.png)

Toolbar layout ở góc phải dưới gồm layout selector, share layout, import layout và arrange/grid controls.

| Layout action | Kết quả |
|---|---|
| Chọn layout | Load layout dashboard đã lưu. |
| Create hoặc duplicate layout | Tạo layout entry mới cho user. |
| Rename layout | Đổi tên layout hiện tại inline. |
| Delete layout | Xoá layout đã lưu đang chọn. |
| Share layout | Copy layout definition hiện tại để dùng lại. |
| Import layout | Mở dialog import JSON và validate trước khi apply. |
| Auto arrange / reset | Sắp xếp lại hoặc restore widget dashboard. |

## Order Ticket

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/dashboard-order-ticket.png" alt="Dashboard order ticket" />
  </div>
  <div className="guide-split-copy">

Ticket nhập lệnh chứa các field account, symbol, order type, side, price, quantity, order summary và buying power preview.

| Ticket action | Kết quả |
|---|---|
| Account / symbol search | Cập nhật ticket context và market preview. |
| Buy / Sell | Đổi side và tính lại order summary. |
| Order type | Đổi nhóm price field cần nhập. |
| Quantity stepper | Tăng hoặc giảm staged quantity. |
| Submit | Gửi ticket khi các field bắt buộc hợp lệ. |
| Cancel / reset | Xoá staged ticket state mà không submit. |

  </div>
</div>

## Price Depth

<div className="guide-split">
  <div className="guide-split-media">

  </div>

Price Depth hiển thị best bid/ask, spread, các mức bid/ask và trade log của symbol đang chọn.
