## Controls

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/price-board-controls.png" alt="Price Board controls" />
  </div>
  <div className="guide-split-copy">

Thanh điều khiển chứa market tabs, filter symbol, thống kê ceiling/up/down/floor và chọn columns.

| Control | Hành vi |
|---|---|
| Watchlist buttons | Load board hoặc submenu group đang chọn và reset row/detail selection. |
| Symbol filter | Lọc board hiện tại theo symbol hoặc name và đưa scroll về đầu. |
| Breadth counters | Tổng hợp số mã ceiling, tăng, tham chiếu, giảm và floor trong filtered set. |
| Arrow keys | Di chuyển selected row lên/xuống và giữ row trong vùng nhìn thấy. |
| Row click | Chọn symbol và mở detail drawer. |
| Header sort | Sort theo các column hỗ trợ như symbol, match price, total volume, change, high/low, foreign flow và CVD. |

  </div>
</div>

## Columns menu

  <div className="guide-split-copy">

Menu `Columns` điều khiển các nhóm cột như ref price, 1-minute session chart, CVD, ceiling/floor, bid levels, symbol và match price.

Switch column dùng để show/hide các column có thể cấu hình. Symbol column bị lock để luôn hiển thị. Nút move up/down đổi thứ tự column, còn `Default` khôi phục layout mặc định.

## Symbol detail drawer

Khi chọn row, detail drawer của symbol được mở. Drawer có thể đóng bằng nút close, backdrop hoặc phím `Escape`.

| Tab | Nội dung |
|---|---|
| Overview | Price depth, price range, session chart, CVD và thông tin ref/ceil/floor. |
| Trade log | Match price trend, session summary và trade tape khi có live tape data. |
| Analysis | Summary chart, financials và money-flow panels. |
| Recommendations | Recommendation và flow views. |
| AI Advisor | Advisor panel dạng chat cho symbol đang chọn. |
