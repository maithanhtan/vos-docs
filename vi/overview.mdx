## Trình tự tham chiếu

<Steps>
  <Step title="Authentication">
    Gọi terminal login API và dùng JWT trả về làm `Authorization: Bearer <jwt>` cho REST API.
  </Step>
  <Step title="Tài liệu API">
    Xem tài liệu REST API, Streaming API và FIX API để kiểm tra các hợp đồng và quy trình được hỗ trợ.
  </Step>
</Steps>
