## Phạm vi hiện tại

| Nhóm | Tài liệu | Mục đích |
|---|---|---|
| Authentication | `POST /api/auth/login` trên `platform.verolabs.co` | Mô tả login terminal và JWT bootstrap. |
| REST API | Public endpoints qua `api-oms.verolabs.co` | Mô tả endpoint đang expose, params, request body, response schema và role gate. |
| Streaming API | Streaming transports và channel messages qua `streaming-oms.verolabs.co` | Mô tả connection endpoints, channel names và live payload structures. |

## Trình tự tham chiếu

<Steps>
  <Step title="Authentication">
    Gọi terminal login API và dùng JWT trả về làm `Authorization: Bearer <jwt>` cho REST API.
  </Step>
  <Step title="REST và streaming APIs">
    Xem REST API và Streaming API để kiểm tra request parameters, response schemas, channel names và live payloads.
  </Step>
</Steps>
