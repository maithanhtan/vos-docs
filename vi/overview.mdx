VOS Docs là bộ tài liệu được duy trì cho vận hành giao diện Vero OMS và tích hợp API v2.
Tài liệu bao gồm các màn hình vận hành, ảnh tham chiếu theo component và endpoint trích xuất CCCD trong luồng đăng ký.

## Phạm vi hiện tại

| Nhóm | Tài liệu | Mục đích |
|---|---|---|
| API v2 | Trích xuất CCCD trong đăng ký | Mô tả contract request, response và lỗi. |
| Hướng dẫn giao diện | Các màn hình chính và luồng vận hành | Mô tả cách người dùng di chuyển trong terminal. |
| Bộ ảnh giao diện | Ảnh page và component ở kích thước MacBook Pro | Cung cấp tham chiếu trực quan cho layout và trạng thái component. |

## Trình tự tham chiếu

<Steps>
  <Step title="Vận hành giao diện">

  </Step>
  <Step title="Contract API v2">
