<Steps>
  <Step title="Enter your account email">
    Enter the email associated with the Vero OMS account, then select **Send Code**.
  </Step>
  <Step title="Check your inbox">
    Retrieve the one-time recovery code sent to the account email.
  </Step>
  <Step title="Enter the recovery code">
    Enter the code and select **Verify & Continue**.
  </Step>
  <Step title="Continue to Vero OMS">
    A valid code restores access and opens the application workspace.
  </Step>
</Steps>
