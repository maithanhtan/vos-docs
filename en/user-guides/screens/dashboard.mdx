Dashboard is the primary trading workspace. It consolidates order entry, market depth, chart context, and the order blotter.

## Full page

![Dashboard full page](/images/vero-oms-ui/dashboard-full.png)

## New Order drawer

`NEW ORDER` opens a right-side drawer containing account, symbol, chart preview, order type, side, price, quantity, order summary, and buying power preview.

![New order drawer](/images/vero-oms-ui/new-order-drawer.png)

## Order Ticket

The ticket contains account, symbol, order type, side, price, quantity, order summary, and buying power preview fields.

![Dashboard order ticket](/images/vero-oms-ui/dashboard-order-ticket.png)

## STOP order mode

When `STOP` is selected, the ticket displays trigger fields such as `WHEN PRICE` and `CONDITION PRICE`. A license warning is displayed when the feature is unavailable.

![Dashboard stop order ticket](/images/vero-oms-ui/dashboard-order-ticket-stop.png)

## BRACKET order mode

When `BRACKET` is selected, the ticket displays `TAKE PROFIT` and `STOP LOSS` fields while retaining summary and buying power context.

![Dashboard bracket order ticket](/images/vero-oms-ui/dashboard-order-ticket-bracket.png)

## Price Depth

Price Depth displays best bid/ask, spread, bid/ask levels, and trade log for the selected symbol.

![Dashboard market depth](/images/vero-oms-ui/dashboard-market-depth.png)

## Price Ladder mode

The Price Depth header includes `PRICE DEPTH` and `PRICE LADDER` display modes, symbol selector, and options menu.

![Dashboard price ladder mode](/images/vero-oms-ui/dashboard-price-ladder.png)

## Trading Chart

The chart component displays candles, timeframe controls, price region, and volume for the selected symbol.

![Dashboard trading chart](/images/vero-oms-ui/dashboard-trading-chart.png)

## TradingView mode

The chart supports `CANDLE` and `TRADINGVIEW` display modes. TradingView mode reserves the chart area for the embedded chart widget.

![Dashboard TradingView mode](/images/vero-oms-ui/dashboard-tradingview.png)

## Order Blotter

The Dashboard blotter filters by status, side, account, and date/time, then displays live order states.

![Dashboard order blotter](/images/vero-oms-ui/dashboard-order-blotter.png)
