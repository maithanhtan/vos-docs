## Layout controls

![Layout toolbar](/images/vero-oms-ui/layout-toolbar.png)

The bottom-right layout toolbar includes layout selection, share layout, import layout, and arrange/grid controls.

| Layout action | Result |
|---|---|
| Select layout | Loads the saved dashboard arrangement. |
| Create or duplicate layout | Adds a new layout entry for the user. |
| Rename layout | Updates the active layout label inline. |
| Delete layout | Removes the selected saved layout. |
| Share layout | Copies the current layout definition for reuse. |
| Import layout | Opens a JSON import dialog and validates the pasted layout before applying it. |
| Auto arrange / reset | Reflows or restores dashboard widgets. |

## Order Ticket

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/dashboard-order-ticket.png" alt="Dashboard order ticket" />
  </div>
  <div className="guide-split-copy">

The ticket contains account, symbol, order type, side, price, quantity, order summary, and buying power preview fields.

| Ticket action | Result |
|---|---|
| Account / symbol search | Updates the ticket context and refreshes market preview values. |
| Buy / Sell | Switches side and recalculates the order summary. |
| Order type | Changes the price fields required by the selected type. |
| Quantity stepper | Increases or decreases the staged quantity. |
| Submit | Sends the staged ticket only when required fields are valid. |
| Cancel / reset | Clears staged ticket state without submitting. |

</div>

## Price Depth

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/dashboard-market-depth.png" alt="Dashboard market depth" />
  </div>
  <div className="guide-split-copy">

Price Depth displays best bid/ask, spread, bid/ask levels, and trade log for the selected symbol.

The header includes a link toggle for syncing the widget symbol, a symbol selector, and an options menu. The options menu controls whether the trade log is visible.

## Trading Chart

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/dashboard-trading-chart.png" alt="Dashboard trading chart" />
  </div>
  <div className="guide-split-copy">

The chart component displays candles, timeframe controls, price region, and volume for the selected symbol.

Changing the symbol resets the chart context. Timeframe controls change the candle interval, and the load-more control extends the visible history when available.

  </div>
</div>

## Order Blotter

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/dashboard-order-blotter.png" alt="Dashboard order blotter" />

  <div className="guide-split-copy">

The Dashboard blotter filters by status, side, account, and date/time, then displays live order states.

Rows can be selected for inspection. Sortable headers reorder the visible set, and the same blotter action model is documented in [Orders](/en/user-guides/screens/orders).
