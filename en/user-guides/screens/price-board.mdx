## Controls

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/price-board-controls.png" alt="Price Board controls" />
  </div>
  <div className="guide-split-copy">

The control bar contains market tabs, symbol filter, ceiling/up/down/floor statistics, and column selection.

| Control | Behavior |
|---|---|
| Watchlist buttons | Loads the selected board or submenu group and resets row/detail selection. |
| Symbol filter | Narrows the selected board locally by symbol or name and resets scroll to the top. |
| Breadth counters | Summarize ceiling, advancing, flat, declining, and floor symbols for the current filtered set. |
| Arrow keys | Move the selected row up or down and keep it in view. |
| Row click | Selects the symbol and opens the detail drawer. |
| Header sort | Sorts by supported single-cell columns such as symbol, match price, total volume, change, high/low, foreign flow, and CVD. |

  </div>
</div>

## Columns menu

<div className="guide-split guide-split--compact">
  <div className="guide-split-media">

  </div>
  <div className="guide-split-copy">

The `Columns` menu controls column groups such as reference price, 1-minute session chart, CVD, ceiling/floor, bid levels, symbol, and matched price.

Column switches show or hide configurable columns. The symbol column is locked so it remains visible. Move up/down controls reorder columns, and `Default` restores the default column layout.

## Symbol detail drawer

Selecting a row opens the detail drawer. The drawer can be closed with the close button, the backdrop, or `Escape`.

| Tab | Contents |
|---|---|
| Overview | Price depth, price range, session chart, CVD, and reference/ceiling/floor information. |
| Trade log | Match price trend, session summary, and trade tape when live tape data is available. |
| Analysis | Summary chart, financials, and money-flow panels. |
| Recommendations | Recommendation and flow views. |
| AI Advisor | Chat-style advisor panel for the selected symbol. |
