Price Board supports symbol monitoring across market groups, bid/ask depth comparison, and visible-column configuration.

## Full page

![Price Board full page](/images/vero-oms-ui/price-board-full.png)

## Controls

The control bar contains market tabs, symbol filter, ceiling/up/down/floor statistics, and column selection.

![Price Board controls](/images/vero-oms-ui/price-board-controls.png)

## Columns menu

The `Columns` menu controls column groups such as reference price, 1-minute session chart, CVD, ceiling/floor, bid levels, symbol, and matched price.

![Price Board columns menu](/images/vero-oms-ui/price-board-columns-menu.png)

## Price Table

The price table displays bid, matched, ask, price, and foreign data for symbols in the selected group.

![Price Board table](/images/vero-oms-ui/price-board-table.png)
