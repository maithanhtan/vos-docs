## PT Workspace Filters

<div className="guide-split">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/put-through-filters.png" alt="Put Through filters" />
  </div>
  <div className="guide-split-copy">

The filter sidebar contains symbol, side, status, account, and IOI ID controls.

| Filter action | Behavior |
|---|---|
| Symbol / side / status / account / IOI ID | Updates the draft filter set in the sidebar. |
| Matching IOIs | Shows how many advertisements match the draft filter before applying it. |
| Apply Changes | Applies the draft filters to advertisements and PT orders. |
| Clear All | Resets the workspace filters to the default state. |

  </div>
</div>

## Advertisements

<div className="guide-split guide-split--wide">
  <div className="guide-split-media">
    <img src="/images/vero-oms-ui/put-through-advertisements.png" alt="Put Through advertisements" />
  </div>
  <div className="guide-split-copy">

The advertisements area displays status tabs, open IOI count, matched count, and notional context.

| Advertisement action | Behavior |
|---|---|
| Status segmented control | Filters the advertisement board by all, active, matched, or rejected states. |
| Header sort | Sorts visible IOIs by id, time, symbol, side, quantity, price, transaction type, status, or party. |
| Row click | Selects the advertisement and pre-fills the PT order form. |
| Pagination | Moves through the filtered advertisement list. |
| New Advertise | Opens the IOI advertisement dialog with symbol, side, quantity, price, contact, market snapshot, notional, and validation state. |

The IOI advertisement dialog sends only after symbol, quantity, and price are valid. Picking a market snapshot price fills the price field.

  </div>
</div>

## PT order form

The right-side form can be filled manually or seeded from a selected advertisement.

| Field / action | Behavior |
|---|---|
| Clear | Removes the linked advertisement and returns the form to fresh state. |
| Account / symbol | Sets the route and instrument. Symbol selection updates market snapshot and account summary. |
| Buy / Sell | Sets quote side and recalculates notional. |
| Quantity / price | Accept formatted numeric input and drive validation. |
| Firm ID / Trader ID | Requires a valid firm and trader pair from reference data. |
| Place PT Quote | Sends the quote request only when account, symbol, quantity, price, and counterparty ids are valid. |
