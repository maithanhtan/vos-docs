The global shell is visible across the Vero OMS workspace. It contains navigation shortcuts, account context, command entry, settings, locale controls, connection status, and layout controls.

## Top navigation and command bar

![Global top navigation](/images/vero-oms-ui/global-top-navigation.png)

The top bar contains the logo, `F1` to `F8` workspace shortcuts, Help, Settings, Fullscreen, display controls, language, clock/time zone, command prompt, New Order, and active context.

| Control | Purpose |
|---|---|
| Documented workspace shortcuts | Switch between Dashboard, Orders, Portfolios, Put Through, Price Board, Operation, and System. |
| `CMD>` | Search or command entry for account and group context. |
| `NEW ORDER` | Opens the order drawer without leaving the current workspace. |
| Help | Opens external help content. |
| Settings | Opens terminal configuration. |
| Fullscreen/display | Adjust display presentation. |
| Language | Switch between `EN` and `VI`. |
| Time zone | Change the time zone used by clock and timestamps. |

## Command search

The `CMD>` field searches account and group context. Selecting an account or group updates the active workspace scope shown beside `NEW ORDER`; the drawer, blotters, portfolios, and risk views then read from that selected context.

| Action | Result |
|---|---|
| Type account, name, or group text | Opens scoped suggestions for account or group selection. |
| Select an account | Sets the active account context for trading and review pages. |
| Select a group | Sets the active group context and updates the `SHOWING GRP` indicator. |
| Press the New Order button | Opens the order drawer from the current page without changing the selected workspace. |

## Settings

The Settings panel has `Profile`, `System`, `Hotkeys`, and `Notifications` tabs. The System tab contains theme, accent color, UI scale, density, broker ticker source, and clear cache controls.

| Setting area | Behavior |
|---|---|
| Profile | Shows identity, current session, active sessions, account context, and sign-out controls. |
| System | Updates theme, accent color, UI scale, density, ticker source, and local display preferences. It also contains the local cache reset action. |
| Hotkeys | Lists configurable shortcuts, identifies conflicts, and supports reset to defaults. |
| Notifications | Controls notification channels, event types, presentation, and sound preferences. |
| Close / backdrop / Escape | Closes the drawer without navigating away from the current page. |
