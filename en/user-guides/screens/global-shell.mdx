The global shell is visible across the Vero OMS workspace. It contains navigation shortcuts, account context, command entry, settings, locale controls, connection status, and layout controls.

## Top navigation and command bar

The top bar contains the logo, `F1` to `F8` workspace shortcuts, Help, Settings, Fullscreen, display controls, language, clock/time zone, command prompt, New Order, and active context.

![Global top navigation](/images/vero-oms-ui/global-top-navigation.png)

| Control | Purpose |
|---|---|
| `F1` to `F8` | Switch between Dashboard, Orders, Portfolios, Algo, Put Through, Price Board, Operation, and System. |
| `CMD>` | Search or command entry for account and group context. |
| `NEW ORDER` | Opens the order drawer without leaving the current workspace. |
| Help | Opens external help content. |
| Settings | Opens terminal configuration. |
| Fullscreen/display | Adjust display presentation. |
| Language | Switch between `EN` and `VI`. |
| Time zone | Change the time zone used by clock and timestamps. |

## Settings

The Settings panel has `Profile`, `System`, `Hotkeys`, and `Notifications` tabs. The System tab contains theme, accent color, UI scale, density, broker ticker source, and clear cache controls.

![Settings system panel](/images/vero-oms-ui/settings-system-panel.png)

## Language menu

The language menu controls the interface language.

![Language menu](/images/vero-oms-ui/language-menu.png)

## Time zone menu

The time zone menu provides search and supported zones for terminal clock and timestamp display.

![Time zone menu](/images/vero-oms-ui/timezone-menu.png)

## Algo menu

`F4 Algo` opens the algo navigation menu. The menu contains Execution Algo, MM Algo, Basket Trade, Neutral Algo, Strategy Algo, and Options Strategy.

![Algo menu](/images/vero-oms-ui/algo-menu.png)

## Status bar

The status bar displays streaming connection, RTT/P99, market ticker, session timer, build version, and layout controls.

![Global status bar](/images/vero-oms-ui/global-status-bar.png)

## Layout controls

The bottom-right layout toolbar includes layout selection, share layout, import layout, and arrange/grid controls.

![Layout toolbar](/images/vero-oms-ui/layout-toolbar.png)

The layout menu displays the active layout, existing layouts, new layout creation, and layout duplication.

![Layout menu](/images/vero-oms-ui/layout-menu.png)
