System provides latency, service state, user access, and audit activity views.

## Full page

![System full page](/images/vero-oms-ui/system-full.png)

## Latency Overview

The latency overview contains filters, latency KPIs, latency vs throughput chart, and system load.

![System latency overview](/images/vero-oms-ui/system-latency-overview.png)

## Latency Table

The latency table lists order traces with gateway, risk, router, venue RTT, total latency, and status.

![System latency table](/images/vero-oms-ui/system-latency-table.png)

## Order Latency Detail

The right-side detail panel displays total latency, bottleneck, stage duration, trace metadata, and order pipeline.

![System order latency detail](/images/vero-oms-ui/system-order-detail.png)

## Users

Users displays user filters, user table, access overview, login history, and selected user panel. The service unavailable state is documented as an error state.

![System users](/images/vero-oms-ui/system-users.png)

## Audit

Audit displays audit trail, filter/search controls, refresh, and the selected audit detail panel.

![System audit](/images/vero-oms-ui/system-audit.png)
