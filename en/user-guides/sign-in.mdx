Use the sign-in page to start an authorized Vero OMS session.

![Vero OMS sign-in page](/images/vero-oms-ui/auth-sign-in.png)

## Sign-in procedure

<Steps>
  <Step title="Enter your email">
    Enter the email associated with your Vero OMS account.
  </Step>
  <Step title="Enter your password">
    Enter the current password. Use the visibility control only when the screen is private.
  </Step>
  <Step title="Select Sign In">
    The button shows a working state while the request is processed. A successful sign-in opens the requested workspace or Dashboard.
  </Step>
</Steps>

## Controls and states

| Control or state | Behavior |
|---|---|
| Email | Requires a valid email value. |
| Password | Requires the current account password. |
| Show / Hide password | Changes only the field display. |
| Sign In | Starts authentication when the required fields are present. |
| Error message | Indicates that access could not be completed. Verify the credentials and try again. |
| Forgot password? | Opens the [Account recovery](/en/user-guides/account-recovery) page. |
