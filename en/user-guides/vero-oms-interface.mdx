Vero OMS is the operator workspace for order entry, order monitoring, portfolio review, market monitoring, operational control, and administration.
The interface uses a shared navigation shell, account context, status indicators, and role-based controls across all workspaces.

## Workspace groups

| Group | Workspaces | Purpose |
|---|---|---|
| Trader | Dashboard, Orders, Portfolios, Put Through, Price Board | Supports trading, market review, order monitoring, and portfolio oversight. |
| Operation | Accounts, Risk Rules, Approvals | Supports account administration, governed changes, and review workflows. |
| Admin | Latency, Users, Audit | Supports operational monitoring, user access, and activity review. |

## Operating context

- Confirm the active account or group before reviewing data or preparing an action.
- Review required fields, validation messages, and confirmation dialogs before submitting a change.
- Use the page status, selected record, and activity history to confirm the current state.
- Available controls depend on the current user's assigned access.
